# dwm Keybindings

The modifier key (`MODKEY`) is set to **Mod4** (usually the Super/Windows key).

| Keybinding | Function |
|------------|----------|
| `MODKEY + Shift + Return` | Spawn terminal (`st`) |
| `MODKEY + Return` | Zoom / Promote to master |
| `MODKEY + p` | Spawn `dmenu` |
| `MODKEY + d` | Decrease nmaster |
| `MODKEY + Shift + d` | Spawn `dmenu_run_desktop` |
| `MODKEY + w` | Spawn browser (`zen-browser`) |
| `MODKEY + Shift + x` | Spawn power menu (`powermenu`) |
| `MODKEY + b` | Toggle status bar |
| `MODKEY + j` | Focus next window in stack |
| `MODKEY + k` | Focus previous window in stack |
| `MODKEY + i` | Increase number of windows in master area |
| `MODKEY + h` | Decrease master area size |
| `MODKEY + l` | Increase master area size |
| `MODKEY + Tab` | View last tag |
| `MODKEY + q` | Kill focused window |
| `MODKEY + t` | Set layout to **Tiled** |
| `MODKEY + f` | Set layout to **Floating** |
| `MODKEY + m` | Set layout to **Monocle** |
| `MODKEY + space` | Set layout to last |
| `MODKEY + Shift + space` | Toggle floating on focused window |
| `MODKEY + 0` | View all tags |
| `MODKEY + Shift + 0` | Tag window with all tags |
| `MODKEY + ,` | Focus previous monitor |
| `MODKEY + .` | Focus next monitor |
| `MODKEY + Shift + ,` | Tag window to previous monitor |
| `MODKEY + Shift + .` | Tag window to next monitor |
| `MODKEY + [1-9]` | View tag 1-9 |
| `MODKEY + Ctrl + [1-9]` | Toggle view tag 1-9 |
| `MODKEY + Shift + [1-9]` | Tag window with tag 1-9 |
| `MODKEY + Ctrl + Shift + [1-9]` | Toggle tag window with tag 1-9 |
| `MODKEY + Shift + q` | Quit dwm |
| `Ctrl + .` | Increase volume |
| `Ctrl + ,` | Decrease volume |
